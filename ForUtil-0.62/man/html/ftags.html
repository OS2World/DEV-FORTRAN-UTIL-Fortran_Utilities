NAME

       ftags - generate a vi tags file for f77 files.

SYNOPSIS

       ftags fortran-files

DESCRIPTION

       Generates  a  case-insensitive  tags file of given fortran
       files for use with vi(1)

COMPATIBILITY

       ftags has run under HP-UX 9 and 10, SGI IRIX  5.2  &  5.3,
       DEC  OSF1,  SunOs  4.1 and 4.3, Cray Unicos, IBM AIX 3.2 &
       4.1, Linux and MSDOS 6.2.  ftags also runs in  a  dos  box
       under Windows 3.1, 3.11 and Win95.
